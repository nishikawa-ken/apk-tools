# APK Fingerprint 取得ツール

APKファイルからフィンガープリント等の情報を取得するための社内ウェブツールです。
Flask + Gunicorn の構成で動作しています。

---

## 起動・停止の手順

ツールを動かしているサーバー（Mac/Linux）のターミナルで、以下の手順を実行してください。

# 1. 仮想環境（.venv）の有効化
プロジェクトのルートディレクトリに移動し、Pythonの仮想環境を有効にします。

```bash
source .venv/bin/activate
```

# 2. ツールの起動（バックグラウンド実行）
Gunicornを使用して、バックグラウンド（裏側）でツールを起動します。
他のPC（社内LAN内）からもアクセスできるよう、ポート 8080 で開放しています。

```bash
gunicorn -D -b 0.0.0.0:8080 get_fingerprint:app
```

* オプション解説:
  * -D : ターミナルを閉じても裏で動き続けるようにします（デーモン化）。
  * -b 0.0.0.0:8080 : 同じネットワーク内のすべてのPCからのアクセスを許可します。

# 3. ツールの停止（終了）
ツールを停止したい、または再起動のために一度落としたい場合は、ポート 8080 を狙い撃ちしてプロセスを終了します。

```bash
kill -9 $(lsof -t -i:8080)
```

## 動作確認・利用方法

# ブラウザからのアクセス方法
ツールが起動したら、同じ社内ネットワーク（LAN）にあるPCのブラウザから以下のアドレスでアクセスできます。

```text
http://[サーバーのIPアドレス]:8080
```
*(例: [http://192.168.1.](http://192.168.1.)XX:8080)*

# 現在動いているか確認したいとき
裏でGunicornが正しく動いているか確認するには、以下のコマンドを叩いてください。

```bash
ps -ef | grep gunicorn
```
意図したプロセスが表示されれば正常に稼働しています。
